💡 CNAME 解析关键点
CNAME 会触发新的 DNS 查询循环，直到遇到 A 记录 或 AAAA 记录 为止。
解析结果会被缓存（TTL），后续请求直接从缓存返回。

图例说明

发起方：用户浏览器发起 DNS 查询请求
DNS 服务器：根服务器、TLD 服务器、权威 DNS
CNAME 记录：域名别名重定向
A/AAAA 记录：域名到 IP 的最终映射
目标服务器：最终响应用户请求的 Web 服务器
